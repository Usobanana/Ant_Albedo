# UI Style Guide & Lock

## 1. 編成画面 (Selection Screen)
- **Grid**: 3列固定 (`repeat(3, 1fr)`)
- **Card**:
    - `background`: #1e293b
    - `border-radius`: 16px
    - `padding`: 12px
    - `text-align`: center
- **Text (Unit Name)**:
    - `font-size`: 11px
    - `font-weight`: 900
    - `white-space`: nowrap (絶対に改行させない)
    - `overflow`: hidden
    - `text-overflow`: ellipsis

## 2. バトル画面 (Battle Screen)
- **Merge Grid**: 5x5 正方形
- **Minimap**: 横幅 90%, 高さ 12px
