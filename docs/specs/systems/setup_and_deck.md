# セットアップ・デッキ編成仕様 (Systems)

## 1. タイトル/セットアップ画面
- **目的**: プレイヤーにステージ情報を提供し、戦略（デッキ）を選択させる。
- **表示内容**:
    - ゲームタイトル
    - 現在のステージ名と出現する敵のプレビュー。
    - ユニット選択パネル。

## 2. デッキ編成ルール
- **選択数**: 全ユニット（5種類以上）の中から 3種類 を選択。
- **バリデーション**: 3種類選ぶまで「ゲーム開始」ボタンは活性化されない。
- **データ共有**: 選択されたユニットの ID は、ゲーム本編のスポーンロジックに引き継がれる。

## 3. ゲームサイクル
- `Setup` (タイトル・編成) → `Action` (パズル・バトル) → `Result` (勝敗・リセット)
