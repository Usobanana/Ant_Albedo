# マージ・スポーンシステム仕様 (スロット独立タイマー版)

## 1. スロット独立スポーン
- **個別タイマー**: 各スロットは独立したクールタイム（例：5秒）を持つ。
- **スポーン条件**: スロットが「空」の状態でのみタイマーが進行する。
- **アイテム出現**: タイマーが100%になると、ランダムなアイテムが出現し、タイマーは停止する。
- **初期状態**: ゲーム開始時、各スロットのタイマーはランダムな進捗（0～90%）から開始され、いくつかのマスには即座にアイテムが配置される。

## 2. マージ（合成）
- **同一合成**: 同じ種類、かつ同じレベルのユニットを重ねることで、レベルが1上昇します。
- **レベル制限**: レベルが上がるほど強力になりますが、Lv0のままでは出撃できません。

## 3. 出撃（デプロイ）
マージされたユニットを戦場へ送り出す方法は2通りあります。

### 3.1 精密デプロイ（ドラッグ＆ドロップ）
- グリッド内のユニットをバトルフィールドへドラッグします。
- **配置自由度**: 自陣（拠点〜最前線）の任意の座標に直接配置できます。

### 3.2 クイック・デプロイ（短押し / タップ）
- **対象**: Lv1以上のユニット。
- **操作**: ユニットを短くタップ（またはクリック）します。
- **結果**: ドラッグの手間を省き、即座に **自拠点（x=150）** へ召喚されます。
- **判定**: 移動距離が 40px 未満の場合に「短押し」と判定されます。
