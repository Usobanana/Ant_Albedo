# マージシステム仕様 (Mechanics)

## 1. グリッド仕様
- **サイズ**: 5x5 (計25マス)
- **配置**: 初期化時に全マスを空 (null) とする。

## 2. アイテム出現
- **間隔**: 3000ms (3秒)
- **対象**: デッキで選択されたユニット系統の Lv0 アイテム。
- **場所**: 現在空いているマスからランダムに1つ。

## 3. マージロジック
- **条件**:
    - ドラッグされたアイテム (Source) と、重なったアイテム (Target) の `type` (系統) が一致。
    - かつ、`level` が一致。
- **結果**:
    - `Target` の `level` が +1 される。
    - `Source` のマスは空になる。
- **上限**: 現在は Lv5 を最大想定。

## 4. UI/フィードバック
- **ハイライト**: ドラッグ中にマージ可能なマスに重なった際、背景を強調表示する。
