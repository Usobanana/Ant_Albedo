# マージ・スポーンシステム仕様 (スロット独立タイマー版)

## 1. スロット独立スポーン
- **個別タイマー**: 各スロットは独立したクールタイム（例：5秒）を持つ。
- **スポーン条件**: スロットが「空」の状態でのみタイマーが進行する。
- **アイテム出現**: タイマーが100%になると、ランダムなアイテムが出現し、タイマーは停止する。
- **初期状態**: ゲーム開始時、各スロットのタイマーはランダムな進捗（0～90%）から開始され、いくつかのマスには即座にアイテムが配置される。

## 2. 視覚演出
- **Circular Gauge**: スロットの背景に円形のゲージを表示。
- **Pop-in**: アイテム出現時は「ポムッ」というアニメーションと共に現れる。

## 3. ミニマップ
- **役割**: 広大な戦場の俯瞰とカメラ制御の可視化。
- **表示内容**:
    - **Hero/King**: 固定アイコン。
    - **Units/Enemies**: 青/赤のドット。
    - **Camera View**: 現在表示されている範囲を枠（Rectangle）で表示。
