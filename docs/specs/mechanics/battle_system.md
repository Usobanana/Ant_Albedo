# バトルシステム仕様 (Mechanics)

## 1. バトルフィールド構成
- **全体**: 横スクロールのシングルライン。
- **左端**: プレイヤー拠点 (Player Base)。
- **右端**: 敵出現ポイント / 敵拠点 (Enemy Base)。

## 2. ユニット出撃 (Deploy)
- **条件**: Lv1 以上のユニットを、上部バトルフィールドエリアへドロップする。
- **制限**: Lv0 アイテムは出撃不可。
- **配置**: 出撃時、縦軸 (Y座標) はランダムな範囲でばらつきを持たせる。

## 3. 進軍・戦闘ロジック
- **進軍**: 出撃したユニットは、系統ごとの `speed` パラメータに基づき右へ移動。
- **攻撃 (現状)**: 右端（敵拠点）に到達した際に、ユニットの `atk` に応じたダメージを敵HPに与え、自身は消滅。
- **戦闘 (Alpha拡張予定)**: 敵ユニットとの衝突判定を行い、足を止めての殴り合いを実装する。

## 4. 勝利・敗北条件
- **勝利**: Enemy HP が 0 になる。
- **敗北**: Player HP が 0 になる。
- **終了時**: アラート表示後、タイトル画面へ戻る。
