# バトルシステム仕様 (V1.1 Alpha)

## 1. 戦場の構造
- **1ライン制**: 全ユニットは単一の進軍路上で戦う。
- **凝縮されたマップ**: ステージ1の初期値は **625px**。ほぼ1画面強に凝縮し、常に激しい接敵が発生する。

## 2. カメラ制御 (Smart Camera)
- **自動追従**: 味方ユニットの「最前線」を追従し、戦況を自動で追う。
- **3秒の余韻 (Wait logic)**: ユーザーが画面を操作（スクロール/タップ）した瞬間、追従を停止。操作終了から **3秒間** は自動追従を再開せず、ユーザーの視点移動を尊重する。

## 3. キャラクターと射程 (Range)
各キャラクターは固有の射程を持ち、ターゲットを捉えると停止して攻撃を開始する。
- **隊列の形成**: 異なる射程を持つユニットを配置することで、自然な軍隊らしい布陣が形成される。
    - **🛡️ナイト (30px)**: 最前線で敵を食い止める壁。
    - **⚔️ソルジャー (40px)**: ナイトの直後から加勢する近接火力。
    - **✨メイジ (120px)**: 中距離から攻撃する魔法職。
    - **🏹アーチャー (150px)**: 後方から安全に狙撃する遠距離職。
- **拠点の自衛 (Hero & King)**: 射程50px内に敵が侵入すると自ら攻撃（ATK 5）を行い、最後の防衛ラインとして機能する。

## 4. 敵の出現（ステージ別ウェーブ）
- **STAGE_CONFIG**: ステージごとに出現間隔、難易度（HP倍率）、最大ウェーブ数が定義される。
- **ステージ1 (はじまりの草原)**:
    - 初期出現間隔: 10秒。
    - 同時出現数: 1体固定（難易度緩和）。
- **激化するウェーブ**: 時間経過とともに、一度に現れる敵の数（最大8体）や出現速度が増加する。

## 5. 配置（デプロイ）
- **領域召喚**: 拠点（x=150）から味方最前線までの範囲にドロップ可能。
- **クイック・デプロイ**: マージグリッド内のLv1以上のユニットを短押し（タップ）すると、即座に拠点（x=150）に召喚される。
