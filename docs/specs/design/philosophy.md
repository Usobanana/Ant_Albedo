# Ant Albedo: Game Design Philosophy

本ドキュメントは、「Ant Albedo」における遊びの質を定義し、バランス調整の指針とするためのものである。

## 1. マージに対する「非線形な報酬」
ユーザーが「マージ」という手間をかける最大の動機は、その先にある圧倒的なパワーの実感である。
- **原則**: ユニットのレベルが上がるごとに、強さは「足し算」ではなく「掛け算」で成長させる。
- **目標**: Lv0を複数出すよりも、リスクを負ってLv2を1体作る方が戦況を覆せる期待値が高い状態を作る。

## 2. ユニットの「極振り」による役割の鮮明化
「どれを出しても同じ」は、戦略の死を意味する。
- **原則**: ステータスを平均化せず、特定の長所を極端に尖らせる。
- **目標**: ユーザーが現在の戦況（「今は耐えるべきか、攻めるべきか」）を判断し、ユニットを選択する意義を最大化する。

## 3. 「緊張感とカタルシス」の波
一定の平坦な戦いは、退屈を生む。
- **原則**: 敵の出現パターンに密度（Wave）を設け、圧倒的な危機（緊張）と、それを精鋭でなぎ倒す瞬間（緩和）のサイクルを作る。
- **目標**: プレイヤーに「ギリギリで耐えきった！」という達成感を与える。

## 4. ユーザー操作の「ダイレクトな反映」
マージ領域と戦場を分断させず、一つの体験として繋げる。
- **原則**: 「自陣召喚」のように、プレイヤーの介入が即座に戦場の有利に結びつくギミックを優先する。
