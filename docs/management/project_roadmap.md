# プロジェクト「Ant_Albedo」開発ロードマップ

## プロジェクト概要
- **ゴール**: 小規模ゲームのリリースとAI開発ナレッジの体系化。
- **主要な役割**:
    - **Project Manager (PM)**: 進行管理、フォルダ構成、ドキュメント管理。
    - **Lead Engineer (LE)**: 技術選定、実装、アーキテクチャ設計。
    - **Product Manager (PdM)**: 各フェーズの承認、品質評価、俯瞰的視点。

| フェーズ | 状態 | 内容 | 担当役割 |

## AI開発ナレッジ管理 (Draft)
※各フェーズで得られた知見を蓄積します。

- **[P1] プロジェクトセットアップ**:
    - 役割の明確化（PM/LE/PdM）により、AIの回答視点を制御する。
    - フォルダ構造の早期定義によるコンテキストの安定化。
- **[P2] AI Role Governance**:
    - 役割分担に基づいたAIへのプロンプト設計および品質管理基準の策定。
